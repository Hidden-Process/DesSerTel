Sistema Tutor

Este sistema evaluará mediante una serie de tests el conocimiento de la materia por parte del estudiante. Mediante estos tests el alumno podrá comprobar su dominio la asignatura. Al final de cada test recibirá una calificación que le servirá como orientación de lo que sabe y se le hará una recomendación cara al examen.

Pablo Bellido (Universidad de Málaga)
